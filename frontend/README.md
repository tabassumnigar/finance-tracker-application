# Frontend

This folder hosts the React + TypeScript + Vite SPA. Pages and features map back to the backend slices.

## Quick orientation
- Entry point: `src/main.tsx` and `src/app/App.tsx`.
- Feature hubs under `src/features` match the requirements (auth, dashboard, transactions, etc.).
- Shared UI, hooks, and store helpers live under `src/components`, `src/hooks`, and `src/store`.
- Assets, types, utils, and routes support cross-cutting concerns.
